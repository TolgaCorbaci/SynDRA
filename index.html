How it works

The same drug, scattered across names and identifiers — reassembled.

SynDRA integrates drug synonyms from KatDB, TTD, PRISM, and LINCS 2020, then anchors every name to a standardized parent structure so that salts, solvates, and stereoisomers of one molecule resolve together.

01

Harmonize names

Synonyms, brand names, and identifiers from four sources are normalized and merged into a single map keyed to L1000 BRD identifiers.

02

Resolve the parent

Each compound is reduced to its salt-stripped parent structure (InChIKey connectivity), so "imatinib mesylate" and "imatinib" become one molecule.

03

Consolidate signatures

Every BRD identifier sharing a parent is linked, so one query recovers all replicate L1000 signatures instead of a single entry.
